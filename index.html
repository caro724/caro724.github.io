Introducción a HTML

HTML es el lenguaje de marcado estándar para crear y estructurar páginas web. Se utiliza para definir el contenido y la organización de elementos como texto, imágenes, enlaces y multimedia mediante etiquetas específicas.

Definición

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). No es un lenguaje de programación, sino de marcado que proporciona la estructura básica de sitios web, interpretada por navegadores como Chrome o Firefo

Cómo funciona

Utiliza etiquetas o tags como "p" para párrafos o "img>"para imágenes, que encierran el contenido. Por ejemplo, "h1>Título h1", crea un encabezado principal. Los navegadores leen este código y lo renderizan visualmente.

Estructura básica

Un documento HTML típico incluye:

. DOCTYPE html: Declara la versión HTML5.

.html: Raíz del documento.

.head: Metadatos (título, charset).

.body: Contenido visible.

Uso con otras tecnologías

HTML se combina con CSS para el diseño visual y JavaScript para interactividad, formando la base de sitios web modernos estáticos o dinámicos.
